# متجر الحيوانات الأليفة - PetStore Lebanon

## تشغيل المشروع محلياً (لتجربته بنفسك)

```bash
cd petstore
pip install -r requirements.txt
python app.py
```

بعدين افتح المتصفح على: http://localhost:5000

أول مرة بتشتغل البرنامج، رح يتنشئ ملف `petstore.db` تلقائياً مع:
- الفئات الخمسة (كلاب، قطط، طيور، أسماك، حيوانات صغيرة)
- ٦ منتجات تجريبية و٢ مقال تجريبي **للمعاينة فقط** (لازم تحذفهم/تعدلهم من لوحة التحكم لما تجهز)
- حساب أدمن مؤقت: `admin` / `changeme123` ⚠️ **لازم تغيّره قبل ما تنشر الموقع فعلياً**

## شغل بدون بيانات تجريبية
لو بدك تبلش بقاعدة بيانات فاضية تماماً (بدون منتجات تجريبية)، احذف ملف `petstore.db` وشغّل:
```bash
python database.py   # بينشئ القاعدة والفئات بس، بدون استدعاء seed_demo_data
```
وبعدين شغل `python app.py` عادي (مش رح يعاود يضيف بيانات تجريبية لأنو الجدول مش فاضي... بصراحة لازم تتأكد إنك ما ناديت seed_demo_data، وهاد بيصير تلقائياً إذا حذفت السطر `seed_demo_data()` من نهاية app.py لما تجهز للنشر الحقيقي).

## النشر على سيرفر Oracle Cloud (Ubuntu 22.04)
السيرفر التطويري المدمج (`app.run`) غير مناسب للإنتاج. الخطوات المختصرة للنشر الحقيقي:
1. `pip install gunicorn`
2. تشغيل التطبيق: `gunicorn -w 3 -b 127.0.0.1:8000 app:app`
3. وضع Nginx كـ reverse proxy أمام gunicorn + شهادة SSL (Let's Encrypt) — هاي خطوة منجهزها سوا أول ما يصير المتجر جاهز للنشر.

## أهم الملفات
- `app.py` — التطبيق والمسارات (routes)
- `database.py` — مخطط قاعدة البيانات + البيانات التجريبية
- `config.py` — الإعدادات (رقم واتساب، اسم المتجر...)
- `templates/` — صفحات HTML
- `static/css/style.css` — التصميم الكامل (Dark Mode)
- `static/js/main.js` — جافاسكريبت (السلة والحاسبات لاحقاً)

## ملاحظة عن الخطوط
الخطوط (Fraunces, Inter, Markazi Text, IBM Plex Sans Arabic) محمّلة من Google Fonts عبر رابط CDN بملف `style.css`. هاد بيحتاج إنترنت عادي وبيشتغل تلقائياً عند فتح الموقع من أي متصفح حقيقي.
